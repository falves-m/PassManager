# PassManager

**PassManager** is a simple and secure password manager for Linux. It allows you to store your passwords in an **encrypted vault**, generate strong passwords, and manage your accounts easily through a graphical interface.  

---

## Features

- Create a **secure vault** to store your passwords.  
- Encrypt passwords with **Fernet** (AES 128-bit symmetric encryption).  
- Manage a **master password** to access the vault.  
- Automatically generate strong passwords.  
- Add, remove, and rename accounts/sites.  
- Quickly search your vault.  
- Double-click an account to copy its password to the clipboard.  
- “**Forgot Password**” option to reset the vault if needed.  

---

## Installation

### Prerequisites

- Python 3.10+  
- Pip  

Just launch the script Build.sh and it should install it automatically. 

If you want to delete it, launch Clean.sh and then delete the folder
